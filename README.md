# Simulador de camara de eventos

Esta es una implementacide un simulador de camara de eventos. Una camara de eventos es un sensor que registrar cambios de intensidad. En contraste con una camara estroboscopica, que captura intensidades de luz incidente sobre cada pixel del sensor de la camara a una tasa constante (30, 60 fps), una camara de eventos captura cambios cambios de intensidad sobre cada pixel en su sensor, y por tanto la tasa de eventos no es constante, sino que depende de la escena que se este capturando.
